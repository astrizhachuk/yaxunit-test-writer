# Соглашения по именованию тестовых модулей

Имя модуля строится по шаблону: `[Префикс_][ИмяОбъекта][_Суффикс]`

## Префиксы (по типу тестируемого объекта)

| Тип объекта              | Префикс | Пример                   |
| ------------------------ | ------- | ------------------------ |
| Общий модуль             | `ОМ_`   | `ОМ_ОбщегоНазначения`    |
| Документ                 | `Док_`  | `Док_ЗаказПокупателя`    |
| Справочник               | `Спр_`  | `Спр_Номенклатура`       |
| Регистр сведений         | `РС_`   | `РС_ЦеныНоменклатуры`    |
| Регистр накопления       | `РН_`   | `РН_ОстаткиНаСкладах`    |
| Регистр бухгалтерии      | `РБ_`   | `РБ_Хозрасчетный`        |
| Регистр расчета          | `РР_`   | `РР_Зарплата`            |
| План счетов              | `ПС_`   | `ПС_Хозрасчетный`        |
| План видов расчета       | `ПВР_`  | `ПВР_Зарплатный`         |
| План видов характеристик | `ПВХ_`  | `ПВХ_Субконто`           |
| План обмена              | `ПО_`   | `ПО_РИБ`                 |
| Обработка                | `Обр_`  | `Обр_ЗакрытиеМесяца`     |
| Отчет                    | `Отч_`  | `Отч_УниверсальныйОтчет` |
| Перечисление             | `Пер_`  | `Пер_СтатусСогласования` |
| Бизнес-процесс           | `БП_`   | `БП_Согласование`        |
| Задача                   | `Зад_`  | `Зад_Задача`             |

## Суффиксы (по типу тестируемого модуля)

| Тип модуля            | Суффикс        | Пример                |
| --------------------- | -------------- | --------------------- |
| Общий модуль          | <Без суффикса> | `ОМ_ОбщегоНазначения` |
| Модуль объекта        | `_МО`          | `Спр_Номенклатура_МО` |
| Модуль менеджера      | `_ММ`          | `Спр_Номенклатура_ММ` |
| Модуль набора записей | `_НЗ`          | `РБ_Хозрасчетный_НЗ`  |

## Примеры составных имен

- `Док_ЗаказПокупателя_ММ` — менеджер документа «Заказ покупателя»
- `РН_ОстаткиНаСкладах_НЗ` — набор записей регистра накопления
- `ПВХ_Субконто_ММ` — менеджер плана видов характеристик
