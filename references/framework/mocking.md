# Мокирование с Мокито

## Три фазы тестирования

1. **Обучение** — настройка поведения методов
2. **Прогон** — выполнение тестируемого кода с активными моками
3. **Проверка** — анализ статистики вызовов

---

## Обучение (Training)

### Точка входа

```bsl
Мокито.Обучение(Объект)
```

`Объект` — общий модуль, менеджер (`Справочники.X`, `Документы.X`, `РегистрыСведений.X`), объект или ссылка.

### Указание метода

```bsl
.Когда("ИмяМетода")                         // по имени строкой
.Когда("ИмяМетода", МассивПараметров)        // с условиями на параметры
.Когда(Модуль.Метод(Парам1, Парам2))         // через реальный вызов (во время обучения вызовы возвращают информацию о вызове)
```

### Реакции

| Метод | Описание |
|-------|----------|
| `.Вернуть(Значение)` | Вернуть значение. Можно цепочкой для последовательных вызовов |
| `.ВыброситьИсключение(Текст)` | Выбросить исключение |
| `.Пропустить()` | Пропустить выполнение метода |
| `.ВыполнитьМетод()` | Выполнить реальный метод (переопределение мока для конкретных условий) |

### Наблюдение

```bsl
.Наблюдать("ИмяМетода")                     // наблюдать без изменения поведения
.Наблюдать("ИмяМетода", МассивПараметров)    // наблюдать с условиями на параметры
```

### Финализация

```bsl
.Прогон()                                    // завершить обучение, активировать моки
```

## Цепочка обучений нескольких объектов

```bsl
Мокито
    .Обучение(Модуль1)
        .Когда("Метод1")
            .Вернуть(Значение1)
    .Обучение(Модуль2)
        .Когда("Метод2")
            .Вернуть(Значение2)
    .Прогон();
```

---

## Цепочка возвратов (последовательные вызовы)

```bsl
Мокито.Обучение(Интеграция)
    .Когда("ВыполнитьЗапрос")
        .Вернуть(1)     // 1-й вызов
        .Вернуть(2)     // 2-й вызов
        .Вернуть(3)     // 3-й и все последующие
    .Прогон();
```

---

## Несколько методов одного объекта

```bsl
Мокито.Обучение(РегистрыСведений.КоэффициентыСезонностиТоварныхЗапасов)
    .Когда("ДобавитьЗапись")
        .Пропустить()
    .Наблюдать("ДобавитьЗапись")
.Обучение(ЦелевыеТоварныеЗапасы)
    .Когда("ПолучитьКоэффициентыСезонности")
        .Вернуть(Коэффициенты)
.Прогон();
```

## Параметры-условия

| Метод | Приоритет | Описание |
|-------|-----------|----------|
| Конкретное значение | 100 | Точное совпадение |
| `Мокито.Параметр(Предикат)` | 90 | Предикатное условие |
| `Мокито.ЧисловойПараметр()` | 10 | Любое число |
| `Мокито.СтроковыйПараметр()` | 10 | Любая строка |
| `Мокито.ТипизированныйПараметр(Тип)` | 10 | Параметр указанного типа |
| `Мокито.ЛюбойПараметр()` | 0 | Любое значение |

При совпадении нескольких условий выигрывает вариант с **наибольшей суммой приоритетов параметров**. При равенстве — первый зарегистрированный.

```bsl
Мокито.МассивПараметров(Парам1, Парам2, ...)  // создать массив параметров для условий
```

**Правила:**
- Без параметров = соответствует ВСЕМ вызовам
- Частичные параметры = проверяются только указанные, остальные игнорируются

## Проверка (Verification)

```bsl
Мокито.Проверить(Объект)
    .КоличествоВызовов("ИмяМетода")      // возвращает число, цепочечный с .Равно(N), .Пусто() и т.д.
    .Вызовы("ИмяМетода")                 // возвращает массив записей вызовов, каждая с .Параметры
```

## Область применения обучения

| Обучение через | Методы менеджера | Методы объекта |
|----------------|------------------|----------------|
| **Менеджер** | Да | Для ВСЕХ объектов этого типа |
| **Объект** | Нет | Только для этого конкретного объекта |
| **Ссылка** | Нет | Для объектов с этой ссылкой |

---

## Ограничения

- Работает только для **методов конфигурации** (через `&Вместо`)
- НЕ может мокировать: встроенные методы платформы, методы расширений, внешних обработок/отчетов
- НЕ может выполнять произвольную логику при вызове метода
- НЕ может модифицировать параметры метода

> Нужно создать новый файл-перехватчик или добавить метод `&Вместо` в существующий? → читай `references/interceptors.md`.

---

## Когда применять Мокито

- Метод обращается к менеджеру справочника/документа/регистра
- Метод вызывает внешний HTTP-сервис
- Нужно проверить количество/параметры вызовов зависимости
- Тест без мока потребовал бы создания реальных данных в БД (медленно, хрупко)

## Когда Мокито не нужен

- Тестируемый метод работает только с переданными параметрами (чистая функция)
- Интеграционный тест, где нужна реальная БД (пометь тест тегом "Интеграционный")

## Реальные примеры из проекта

### Заглушка общего модуля

```bsl
Мокито.Обучение(КолеровкаПовтИсп)
    .Когда("ЭтоКолеровка")
        .Вернуть(Истина)
    .Прогон();
```

### Заглушка менеджера регистра сведений

```bsl
Мокито.Обучение(РегистрыСведений.НастройкиОпределенияУслугиКолеровка)
    .Когда("НайтиУслугуКолеровки")
        .Вернуть(Ожидание)
    .Прогон();
```

### Заглушка менеджера справочника

```bsl
Мокито.Обучение(Справочники.Номенклатура)
    .Когда("НайтиНоменклатуруПоКоду")
        .Вернуть(ОписаниеТоваров)
    .Прогон();
```

### Заглушка менеджера документа

```bsl
Мокито.Обучение(Документы.ЗаказПокупателя, Ложь)
    .Когда("ПолучитьДоступныеОбъектыПроектныхПродажПоОписанию")
        .Вернуть(Новый Массив())
    .Наблюдать("ПолучитьДоступныеОбъектыПроектныхПродажПоОписанию")
    .Прогон();
```

### Наблюдение + проверка кеширования

```bsl
Мокито.Обучение(КолеровкаПовтИсп)
    .Наблюдать("ЭтоКолеровка")
    .Прогон();

КолеровкаПовтИсп.ЭтоКолеровка(Справочники.Номенклатура.Доставка);
КолеровкаПовтИсп.ЭтоКолеровка(Справочники.Номенклатура.Доставка);

Мокито.Проверить(КолеровкаПовтИсп)
    .КоличествоВызовов("ЭтоКолеровка")
        .Равно(1);
```

### Цепочка обучений нескольких объектов с Пропустить и Наблюдать

```bsl
Мокито
    .Обучение(РегистрыСведений.КоэффициентыСезонностиТоварныхЗапасов)
        .Когда("ДобавитьЗапись")
            .Пропустить()
        .Наблюдать("ДобавитьЗапись")
    .Обучение(ЦелевыеТоварныеЗапасы)
        .Когда("ПолучитьКоэффициентыСезонности")
            .Вернуть(Коэффициенты)
    .Прогон();
```

### Множественное обучение одного модуля (разные задачи)

```bsl
Мокито.Обучение(Колеровка)
    .Когда("НайтиУслугуКолеровкиПоВидуКолеровки")
        .Вернуть(Справочники.Номенклатура.ПустаяСсылка())
    .Прогон();

Мокито.Обучение(Колеровка)
    .Наблюдать("НайтиУслугуКолеровкиПоВидуКолеровки")
    .Прогон();
```
