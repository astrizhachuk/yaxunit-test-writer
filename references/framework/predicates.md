# Предикаты YAxUnit

## Точка входа

```bsl
ЮТест.Предикат()
```

Создает новый предикат с текучим интерфейсом. Опционально принимает `Структуру` для автоматического создания предикатов равенства по ключ-значение:

```bsl
Условия = Новый Структура("Наименование, Код", НаименованиеОбъекта, КодОбъекта);
Предикат = ЮТест.Предикат(Условия);
```

---

## Выбор свойства

| Метод | Описание |
|-------|----------|
| `Реквизит(ИмяРеквизита)` | Указывает свойство для последующих проверок. Поддерживает точечную нотацию: `"ВложенныйОбъект.Число"`, `"Товары.Номенклатура"` |
| `Свойство(ИмяСвойства)` | Алиас для `Реквизит` |

Если не указан `Реквизит`/`Свойство` — условие применяется к самому объекту.

---

## Сравнение

| Метод | Описание |
|-------|----------|
| `Равно(Значение)` | Равенство |
| `НеРавно(Значение)` | Неравенство |
| `Больше(Значение)` | Больше |
| `БольшеИлиРавно(Значение)` | Больше или равно |
| `Меньше(Значение)` | Меньше |
| `МеньшеИлиРавно(Значение)` | Меньше или равно |

---

## Состояние

| Метод | Описание |
|-------|----------|
| `Заполнено()` | Заполнено (не пусто) |
| `Пусто()` | Пусто/не заполнено |

---

## Тип

| Метод | Описание |
|-------|----------|
| `ИмеетТип(ИмяТипа)` | Имеет указанный тип |
| `ИмеетТипОтличныйОт(ИмяТипа)` | Тип отличается от указанного |

---

## Длина/размер

| Метод | Описание |
|-------|----------|
| `ИмеетДлину(Значение)` | Длина/размер равна значению |
| `ИмеетДлинуОтличнуюОт(Значение)` | Длина/размер не равна значению |

---

## Наличие свойства

| Метод | Описание |
|-------|----------|
| `ИмеетСвойство(ИмяСвойства)` | Содержит вложенное свойство |
| `НеИмеетСвойства(ИмяСвойства)` | Не содержит вложенного свойства |

---

## Вхождение

| Метод | Описание |
|-------|----------|
| `Содержит(Значение)` | Содержит значение |
| `НеСодержит(Значение)` | Не содержит значение |
| `СодержитСтрокуПоШаблону(Шаблон)` | Строка соответствует регулярному выражению |
| `НеСодержитСтрокуПоШаблону(Шаблон)` | Строка не соответствует регулярному выражению |
| `ВСписке(СписокЗначений)` | Значение входит в указанный список |

---

## Диапазон

| Метод | Описание |
|-------|----------|
| `Между(Начало, Окончание)` / `МеждуВключаяГраницы(Начало, Окончание)` | [Начало, Окончание] |
| `МеждуИсключаяГраницы(Начало, Окончание)` | (Начало, Окончание) |
| `МеждуВключаяНачалоГраницы(Начало, Окончание)` | [Начало, Окончание) |
| `МеждуВключаяОкончаниеГраницы(Начало, Окончание)` | (Начало, Окончание] |

## Получить() — финализация предиката

```bsl
.Получить()
```

Возвращает копию накопленных условий предиката в виде массива. **Критично** при сохранении предиката в переменную или передаче нескольких предикатов — без `.Получить()` предикаты разделяют один глобальный контекст.

### Неправильно (оба предиката указывают на один контекст):

```bsl
УсловиеСтрока = ЮТест.Предикат().ИмеетТип("Строка");
УсловиеЧисло = ЮТест.Предикат().ИмеетТип("Число");
// Оба будут "Число"!
```

### Правильно (независимые копии):

```bsl
УсловиеСтрока = ЮТест.Предикат().ИмеетТип("Строка").Получить();
УсловиеЧисло = ЮТест.Предикат().ИмеетТип("Число").Получить();
```

**Исключение:** `.Получить()` не нужен при передаче предиката непосредственно в метод утверждения или Мокито, если больше нигде не используется.

---

## Контексты использования

1. **Утверждения коллекций** — `.Содержит(Предикат)`, `.ЛюбойЭлементСоответствуетПредикату(Предикат)`, `.КаждыйЭлементСоответствуетПредикату(Предикат)`
2. **Утверждения БД** — `ОжидаетЧтоТаблицаБазы(...).СодержитЗаписи(Предикат)`
3. **Запросы данных** — `ЮТЗапросы.Запись(ИмяТаблицы, Предикат)`
4. **Обучение Мокито** — `Мокито.Параметр(Предикат)`

## Реальные примеры из проекта

### Поиск элемента в Соответствии по ключу и значению

```bsl
ЮТест.ОжидаетЧто(Результат.Товары[Товар.Код])
    .Свойство("Отрезки")
        .ИмеетТип("Соответствие")
        .ИмеетДлину(2)
        .Содержит(
            ЮТест.Предикат()
                .Реквизит("Ключ").Равно(Отрезок1.Длина)
                .Реквизит("Значение").Равно(Новый Структура("Количество, Длина", Отрезок1.Количество, Отрезок1.Длина)))
        .Содержит(
            ЮТест.Предикат()
                .Реквизит("Ключ").Равно(Отрезок2.Длина)
                .Реквизит("Значение").Равно(Новый Структура("Количество, Длина", Отрезок2.Количество, Отрезок2.Длина)));
```

### Проверка массива структур по нескольким свойствам

```bsl
ЮТест.ОжидаетЧто(Результат)
    .ИмеетТип("ФиксированныйМассив")
    .ИмеетДлину(2)
    .Содержит(
        ЮТест.Предикат()
            .Реквизит("Код").Равно(Товары[0].sku)
            .Реквизит("ДлиныОтрезков").Заполнено())
    .Содержит(
        ЮТест.Предикат()
            .Реквизит("Код").Равно(Товары[1].sku)
            .Реквизит("ДлиныОтрезков").Заполнено());
```

### Предикат с несколькими условиями на одном реквизите

```bsl
ЮТест.Предикат()
    .Реквизит("Число")
        .ИмеетТип("Число")
        .Больше(0)
        .Меньше(2)
        .Равно(1)
    .Получить();
```

### Предикат из структуры (сокращенная запись)

```bsl
Отбор = ЮТест.Предикат()
    .Реквизит("Источник").Равно(Объект)
    .Реквизит("ТипОповещения").Равно(Справочники.ТипыОповещенийПользователя.Уведомление)
    .Получить();

ЮТест.ОжидаетЧтоТаблицаБазы(ИмяРегистра).НеСодержитЗаписи(Отбор);
// ... вызов метода ...
ЮТест.ОжидаетЧтоТаблицаБазы(ИмяРегистра).СодержитЗаписи(Отбор);
ДанныеУведомления = ЮТЗапросы.Запись(ИмяРегистра, Отбор);
```
